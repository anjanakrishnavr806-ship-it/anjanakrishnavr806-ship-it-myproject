# Machine Learning Projects

This repository contains multiple machine learning projects implemented using Python.

## Projects

### ECG Prediction (Machine Learning)
- Built a machine learning model to predict normal and abnormal ECG signals.
- Performed data preprocessing, feature extraction, and model training.
- Used classification algorithms for healthcare-related prediction.

### Diabetes Prediction (Machine Learning)
- Developed a diabetes prediction system using Random Forest algorithm.
- Processed medical input data and evaluated model performance.

### Face Mask Detection (Machine Learning)
- Implemented a machine learning-based model to detect face mask usage.
- Trained the model to classify masked and unmasked images.

### House Price Prediction (Machine Learning)
- Built a regression-based machine learning model to predict house prices.
- Analyzed input features to improve prediction accuracy.

### News Classification (Machine Learning)
- Developed a text classification model to categorize news articles.
- Applied text preprocessing and machine learning algorithms.

## Technologies Used
- Python  
- Machine Learning (Scikit-learn)  
- Pandas, NumPy, Matplotlib
